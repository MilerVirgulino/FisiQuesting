# Egg sprites

Coloque aqui os PNGs finais que o app deve carregar no avatar.

O caminho e montado automaticamente a partir da categoria e do `id` cadastrado em `src/data/avatarItems.js`.

Exemplo:

```js
roupa_azul: { label: "Roupa azul" }
```

Arquivo esperado:

```txt
public/assets/egg-sprites/outfits/roupa_azul.png
```

Use PNG transparente em `256x256` px.
